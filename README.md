# Jogo Perfil 100 ∞

Um jogo de adivinhação estilo Perfil, feito em HTML puro, que roda offline no celular e no GitHub Pages.

**Demo:** https://tillimichel-max.github.io

## Como jogar
1. Escolha uma dica (1 a 12)
2. Digite seu palpite
3. Errou? O jogo revela 1 letra por vez até acertar
4. Pontos só caem quando abre dicas, não quando erra

## Recursos
- 100 cartas (Pessoas, Lugares, Coisas, Anos)
- 12 dicas por carta
- Modo infinito
- 2 jogadores com nomes
- Placar salvo no celular (localStorage)
- Som de acerto/erro
- Botões Pular e Revelar
- Fotos reais da Wikipedia (com fallback offline)
- Funciona 100% offline

## Instalar no GitHub Pages
1. Faça fork ou crie repositório `https://detiillimichel-max.github.io/perfil-imagem-/`
2. Envie o arquivo `index.html`
3. Ative Pages em Settings > Pages > Branch main
4. Acesse https://seu-usuario.github.io
